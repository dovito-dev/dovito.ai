# Dovito.ai Landing Page - Comprehensive Development Prompt

Build a modern, interactive landing page for dovito.ai inspired by beam.ai's design aesthetics. The centerpiece will be a "Periodic Table of Apps" showcasing our software products with CMS functionality for easy management and automatic logo generation.

## Core Features & Requirements

### 1. Periodic Table Showcase
- **Layout**: Grid-based periodic table design with product tiles
- **Tile Design**: Square format (either 90-degree corners or rounded)
- **Product Representation**: Each app gets a tile with its abbreviation (Pe for pe.dovito.com, SM for sm.dovito.com, etc.)
- **Interactive Elements**: 
  - Hover effects with product details
  - Click to navigate to product URLs
  - "Coming Soon" states for products in development
  - Smooth transitions and animations

### 2. Content Management System (CMS)
- **Admin Interface**: Simple, intuitive CMS for non-technical users
- **Product Management**:
  - Add/edit/delete products
  - Product name, abbreviation, description, status (live/coming soon)
  - Product URL, category, launch date
  - Upload product images/icons
  - Set periodic table position
- **Content Management**:
  - Hero section content
  - Company information
  - Feature highlights
  - Testimonials/case studies
- **User Authentication**: Secure admin login system

### 3. Automatic Logo Generation
- **Input System**: Upload product information through CMS
- **Logo Generation**: 
  - Auto-create professional product logos based on uploaded info
  - Generate downloadable PNG files
  - Multiple size variants (favicon, social media, print)
  - Consistent brand styling across all generated logos
- **Display**: Showcase generated logos on homepage in a dedicated section

### 5. Technical Requirements

#### Frontend Stack
- **Framework**: Next.js 14 with TypeScript
- **Styling**: Tailwind CSS for responsive design
- **Animations**: 
  - React-bits splash cursor animation (mandatory)
  - Additional smooth transitions and micro-interactions
  - Framer Motion for complex animations
- **Icons**: Lucide React or similar professional icon library

#### Backend & Database
- **Database**: Supabase for CMS data storage
- **Authentication**: Supabase Auth for admin access
- **File Storage**: Supabase Storage for images and generated logos
- **API**: Next.js API routes for CMS operations

#### Logo Generation System
- **Technology**: Canvas API or similar for dynamic logo creation
- **Brand Consistency**: Use uploaded Dovito logo as style reference for all generated product logos
- **Templates**: Professional design templates maintaining Dovito's visual identity
- **Export**: High-quality PNG generation and download functionality
- **Automation**: Trigger generation when product information is updated

#### Performance & SEO
- **Mobile-First**: Fully responsive design
- **Performance**: Optimized images, lazy loading, fast page speeds
- **SEO**: Proper meta tags, structured data, sitemap
- **Accessibility**: WCAG compliance, keyboard navigation, screen reader support

### 6. User Experience Flow

#### Public User Journey
1. Land on homepage with animated hero section
2. Explore periodic table to discover products
3. Click products to learn more or visit live applications
4. View company value proposition and social proof
5. Take action via contact forms or consultation booking

#### Admin User Journey
1. Secure login to CMS dashboard
2. Manage products (add, edit, delete, reorder)
3. Upload product information and trigger logo generation
4. Update homepage content and messaging
5. Preview changes before publishing
6. Download generated logos and assets
