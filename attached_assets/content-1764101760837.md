# Brand Kit Checklist

# Brand Kit Checklist

# 1\. Brand Strategy & Essence

Mission Statement: Why the brand exists. Vision Statement: The long-term aspiration. Core Values: Principles and beliefs guiding brand behavior. Brand Story: Narrative/history that shapes your brand identity. ● Positioning Statement: What differentiates the brand. Audience Personas: Key target audiences or customer profiles. Brand Personality: The human traits of the brand.

# 2\. Logo System

Primary Logo: Main logo in all approved colorways and file formats (PNG, SVG, EPS, PDF).

Secondary/Alternative Logos: Horizontal, vertical, simplified, icon-only options.

Logo Usage Guidelines: ● Minimum clear space/margin ● Minimum size ● Background control (white, color, photographic) Incorrect usages ("don'ts")

Favicon: Web icon version.

Downloadable Files: Organized folder with all logo variations.

# 3\. Color Palette

Primary Colors: Exact HEX, RGB, CMYK, and/or Pantone references.

● Secondary/Accent Colors: Support colors for highlights, accents, backgrounds.

● Gradients: Standardized gradients for backgrounds/text, including CSS code if for web.

● Background/Surface Colors: Used for UI, cards, sections.

● Border Colors: For dividing areas, UI separation.

● Text Colors: For body, headings, muted text.

● Contrast Accessibility Guidelines: Ratios for readability.

# 4\. Typography

Font Families: Official typefaces (headings, subheadings, body, UI). Web-safe Alternatives: Substitution fonts if custom/paid fonts unavailable. Usage Hierarchy: Headings (H1-H6), paragraph, UI labels, buttons. Font Weights & Styles: Bold, regular, italics, etc. Size Examples: px/rem/em for web, pt for print. Line Height, Letter-spacing: For readability and consistent look. ● Licensing Info: Font source and usage rights.

# 5\. Spacing & Layout

Grid System:

Standardized grid, column counts, gutter spacing. Spacing Units:

Padding, margins, measurements for consistency. Safe Area Guidelines:

Margins for logos, buttons, key elements.

Responsive Layout Rules:

Mobile, tablet, desktop guidelines.

# 6\. Visual Elements

Imagery Style:

Guidance for photography (color, B&W, filters, moods). Illustrations:

Approved style and usage (vector, icon, mascot).

Iconography:

Brand-approved icon sets, sizes, examples.

Patterns & Textures:

Decorative graphics, backgrounds.

Graphics/Shapes:

Branded elements/motifs for use in content.

# 7\. Components & UI Elements

Buttons:

Primary/secondary button styles (rest, hover, pressed). Form Inputs:

Text fields, dropdowns, toggles.

Cards/Containers:

How to display information blocks.

Navigation Bars, Footers:

Standard layouts and components.

Shadows, Elevation:

Depth guidance for UI.

# 8\. Voice & Tone

Brand Voice:

Description (formal, friendly, technical, playful).

Tone Guidelines:

Different contexts (social, support, sales).

Sample Messaging:

Taglines, elevator pitch, mission statement, catchphrases.

# 9\. Application Examples

Stationery:

Business cards, letterheads, envelopes. Email Signatures:

Samples and setup instructions. Social Media Templates:

Example post layouts, cover images, profile photos. Marketing Assets:

PowerPoint/Keynote templates, brochures, one-pagers. Merchandise Examples:

T-shirts, packaging, swag.

# 10\. Assets Repository & Contact Info

Download Link:

Folder/drive with all logos, fonts, images, components.

Contact Information:

Brand or design team contacts for questions or asset access.

# 11\. Governance & Usage

Approval Process:

Who to contact for asset requests or questions. Do's and Don'ts:

Condensed checklist for quick reference.

Links to Legal Policies:

Trademark, copyright, usage permissions.
